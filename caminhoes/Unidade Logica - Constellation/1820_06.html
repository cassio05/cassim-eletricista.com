Descrição da Falha

Circuito da saída linha 58d, curto circuito para o massa.



Indicação de Falha

Não haverá indicação de falha.


Estratégia de Monitoramento

A Unidade Lógica (LU) monitora o circuito da saída do terminal 58d quanto à curto para o massa e/ou interrupção.

 

Efeito da Falha

Afetará todos os componentes alimentado por este terminal.

 

 

Possíveis Falhas

FMI 5: Circuito aberto, interrompido.

FMI 6: Curto circuito para o massa.

Verifique qual sistema alimentado pelo 58d encontra-se em curto para o massa.
