Módulo PTM Euro V

Código: 5782/04

Circuito Rede CAN Módulo LU (Unidade Lógica)

 


    1 - Tacógrafo Modular.

    2 - Unidade Lógica (LU).

    3 - Painel de Instrumentos.

    4 - PTM (Power Train Manager).

 

Descrição da Falha

Tempo de espera de mensagem da rede CAN do módulo da LU. Tempo de espera acima do limite, sem sinal.


 

Indicação de Falha

Luz de alerta acende constantemente com os símbolos   e   e ativa o alarme sonoro longo, com o veículo andando ou parado.


 

Estratégia de Monitoramento

O módulo  aguarda a recepção de mensagens do módulo LU via CAN, não as recebendo, libera a mensagem de falha.


 

Efeito da Falha

Tacógrafo não operará corretamente.

Não é possível marcação de velocidade (velocímetro não funciona).


 

Possíveis Falhas

FMI 4: Sem sinal. 


 

Aviso

As verificações do módulo TCO  para a PTM são necessárias para diagnosticar possíveis interferências no circuito da LU.



 

Registros de Falhas em Outros Módulos de Comando

Não.


 

Considerar os esquemas elétricos do veículo


Verificação Medição Eliminação da Falha

Módulo TCO (tacógrafo) para PTM (Power Train Manager)
Continuidade do circuito:

- TCO pino A4 para Unidade Lógica (LU) pino B1

- Unidade Lógica (LU) pino B3 para módulo PTM pino B6

- TCO pino A4 para Painel de instrumentos pino X1 A9

- Painel de instrumentos pino X1 A15 para para módulo PTM pino B6

- TCO pino A8 para Unidade Lógica (LU) pino B7

- Unidade Lógica (LU) pino B9 para módulo PTM pino B18

- TCO pino A8 para Painel de instrumentos pino X1 A8

- Painel de instrumentos pino X1 A14 para para módulo PTM pino B18


– Verificar os cabos quanto a rompimento ou oxidação


– Verificar as conexões quanto a mau contato ou pino torto


- Se não for encontrada nenhuma falha, substituir o módulo TCO (Tacógrafo Modular)

Se o dispositivo de diagnóstico MCO apresenta as seguintes falhas:

Verificar conforme a falha 5505_04



- Verificar os cabos quanto a rompimento ou oxidação
