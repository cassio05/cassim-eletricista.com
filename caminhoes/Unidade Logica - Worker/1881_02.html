Unidade Lógica (LU) - Worker Euro V

 

código1881-02

 

Descrição da falha

 

Sensor de nível Vazio do líquido de arrefecimento implausível

(Ex: Sensor de nível baixo indicando que há líquido e o sensor vazio indicando que não há líquido).

 

 

Causa

 

 

Detectado leitura incorreta do sensor de nível vazio do arrefecimento.

 

 

Efeito da falha

 

 

Luz de aviso se acende juntamente com o símbolo PARE.

 

 

Circuito

 

 

 

Legenda

 

A356v Unidade Lógica (LU)
B139 Sensor de nível do arrefecimento baixo
B139v Sensor de nível do arrefecimento vazio
X515-B Conector de interface B (marron)

 

 

Avisos

 

Nota:

Leia sempre as Recomendações para Diagnósticos.

 

Verifique visualmente o circuito dos sensores quanto a fios derretidos descascados ou mau contato nos conectores.

 

 

Passos do diagnóstico de falhas

 

Etapa 01

 

Chave de ignição na posição desligada.

Desconecte as extremidades dos circuitos.
Meça a continuidade entre os terminais D4, D14, D5 e D15 do conector da Unidade Lógica (LU) para o massa, que não deve haver continuidade.

Meça a isolação entre o terminal D4, D14, D5 e D15 do conector da Unidade Lógica (LU) para os demais conectores de interface D (marron).

Está ok?

 

Etapa 02

 

Repare e/ou substitua o chicote elétrico.

Vá para a etapa 09.

 

 

 

 

 

 

 

 
   

Etapa 03

 

Para fins de teste, substitua o sensor de nível baixo por um em bom estado.

 

O sistema opera normalmente?

 

Etapa 04

 

Recoloque o sensor de nível baixo original.

Vá para a etapa 06.

 

 

 
   

Etapa 05

 

Substitua o sensor de nível baixo por um novo.

Vá para a etapa 09.

 

    
   

Etapa 06

 

Para fins de teste, substitua a Unidade Lógica (LU) por um em bom estado.

 

O sistema opera normalmente?

 

Etapa 07

 

Recoloque a Unidade Lógica (LU) original.

Vá para a etapa 01 e refaça os testes..

 

 

 
   

Etapa 08

 

Substitua a Unidade Lógica (LU) por uma nova.

Vá para a etapa 09.

 

    
   

Etapa 09

 

Conecte todos os conectores.

Chave de ignição na posição ligada.

Apague os códigos de falha.

Teste o sistema quanto ao funcionamento normal.

 

O sistema opera normalmente?

Etapa 10

 

Vá para a etapa 01 e refaça todos os testes.

 

 

 

 

 
   

Etapa 11

 

Libere o veículo.
