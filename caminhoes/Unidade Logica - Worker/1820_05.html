Unidade Lógica (LU) - Worker Euro V

 

código1820-05

 

 

Descrição da falha

 

 

Saída da iluminação do painel de instrumentos com circuito aberto.

 

 

Causa

 

 

Interrupção no circuito de saída da iluminação do painel de instrumentos.

 

 

Efeito da falha

 

 

Afeta todos os componentes alimentados por este terminal.

 

 

Circuito

Avisos

 

Nota:

Leia sempre as Recomendações para Diagnósticos.

 

O painel de instrumentos monitora a saída 58d através do pino A17 do conector X2 e envia mensagem para a Unidade Lógica (LU) através

da rede CAN.

 

 

Passos do diagnóstico de falhas

 

Etapa 01

 

Chave de ignição na posição desligada.
Desconecte do Cluster o conector verde 32 vias - Painel de Instrumentos (X2).
Meça a continuidade entre o terminal A17 (X2) do Cluster para todos os componentes conforme o diagrama acima.

Está ok?

 

Etapa 02

 

Repare e/ou substitua os componentes danificados, e vá para a etapa 07.

 

 

 

 

 
   

Etapa 03

 

Para fins de teste, substitua o painel de instrumentos (Cluster).

O sistema opera normalmente?

 

Etapa 04

 

Vá para a etapa 07.

 

 

 
   

Etapa 05

 

Para fins de teste, substitua a Unidade Lógica(LU).

O sistema opera normalmente?

 

 

Etapa 06

 

Recoloque a Unidade Lógica(LU) original.

Vá para a etapa 1 e refaça os testes.

 

 
   

Etapa 07

 

Conecte todos os conectores.

Chave de ignição na posição ligada.

Apague os códigos de falha.

Teste o sistema quanto ao funcionamento normal.

A falha permanece ativa?

 

Etapa 08

 

Volte à etapa 01 e refaça todos os testes.

 

 

 

 

 
   

Etapa 09

 

Libere o veículo.
