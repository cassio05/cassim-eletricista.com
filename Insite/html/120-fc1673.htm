Nível do Tanque de Fluido de Escape de Diesel do Sistema de Pós-tratamento - Dados Válidos, mas Abaixo da Faixa Normal de Operaçăo - Nível Mais Severo. Năo foi detectada a presença de fluido de escape de diesel no tanque de fluido do sistema de pós-tratamento.

A injeçăo de fluido de escape de diesel no sistema de pós-tratamento de SCR será desabilitada. Despotenciamento imediato do motor.

Descriçőes do Circuito

O sensor de nível do tanque de fluido de escape de diesel do sistema de pós-tratamento é usado para monitorar o nível de fluido de escape de diesel no interior do tanque de fluido.

Localizaçăo do Componente

O sensor de nível de fluido de escape de diesel está localizado no tanque de fluido de escape de diesel. A localizaçăo do tanque e do sensor do nível da soluçăo dependem do OEM.

Condiçőes para Execuçăo do Diagnóstico

  • Este diagnóstico é executado continuamente quando a chave de igniçăo encontra-se na posiçăo ON.

Condiçőes para Configurar os Códigos de Falha

  • O sensor de nível do tanque de fluido de escape de diesel detectou que o tanque de fluido está vazio.

Açăo tomada quando o código de falha está ativo

  • O ECM acende a luz âmbar de verificaçăo do motor (CHECK ENGINE) e/ou a luz indicadora de falha (MIL) começa a piscar quando o diagnóstico é executado e falha.
  • Será ativado imediatamente um despotenciamento do motor quando o código de falha for ativado.

Condiçőes para Apagar o Código de Falha

Este código de falha năo pode ser apagado com a ferramenta eletrônica de serviço INSITE™.

  • Para validar o reparo, dę partida no motor e deixe-o funcionar em marcha lenta durante 1 minuto.
  • O ECM apaga a luz âmbar de verificaçăo do motor (CHECK ENGINE) e/ou a luz indicadora de falha (MIL) quando o diagnóstico é executado e é aprovado no teste.
  • Utilizando a ferramenta eletrônica de serviço INSITE™, certifique-se de que o código de falha esteja inativo.
  • O código de falha inativo será apagado da memória após 9600 horas de funcionamento do motor sem nova ocorręncia da mesma falha.

Dicas para a Oficina

O Código de Falha 1673 será acionado quando o sensor de nível do fluido de escape de diesel detectar que o tanque de fluido está vazio. O fluido de escape de diesel ainda poderá ser visto no interior do tanque enquanto este código de falha estiver ativo. Adicione fluido de escape de diesel no tanque.

Consulte o diagnóstico do Código de Falha 1673.
