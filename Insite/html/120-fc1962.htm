Temperatura Excessiva (Calculada) do Acionador do Atuador do TGV - Dados Válidos mas Acima da Faixa Normal de Operaçăo - Nível Menos Severo. Detectada temperatura interna alta do atuador do TGV.

Nenhum quanto ao desempenho.
ISL8.9 CM2150 SN - Circuito do Atuador do TGV

Descriçőes do Circuito

O turbocompressor de geometria variável (TGV) é ativado eletronicamente pelo atuador. O atuador do TGV é um dispositivo 'inteligente' que recebe informaçőes pelo datalink J1939 enviadas pelo módulo eletrônico de controle (ECM) do motor primário. O atuador do TGV realiza seus próprios diagnósticos e relata falhas para o ECM do motor primário, através do datalink J1939.

Localizaçăo do Componente

O atuador do TGV está localizado no alojamento dos rolamentos do turbocompressor.

Condiçőes para Execuçăo do Diagnóstico

Este diagnóstico é executado quando a chave de igniçăo é ligada (ON).

Condiçőes para Configurar os Códigos de Falha

O código de falha será registrado quando a temperatura interna do atuador do TGV exceder 125°C [257°F] continuamente durante 30 minutos.

Açăo tomada quando o código de falha está ativo

  • O ECM acende a luz amarela de verificaçăo do motor (CHECK ENGINE) imediatamente quando o diagnóstico é executado e falha.

Condiçőes para Apagar o Código de Falha

O ECM apaga a luz âmbar de verificaçăo do motor (CHECK ENGINE) imediatamente depois que a temperatura interna do atuador cair abaixo de 125°C [257°F].

Dicas para a Oficina

Este código de falha será desativado depois de arrefecido o atuador do TGV (veículo parado na oficina). Faça o diagnóstico de várias contagens inativas deste código de falha como um código de falha ativo.

Verifique também a existęncia de vazamentos de escape sobre o atuador do TGV. Repare quaisquer vazamentos externos.

Consulte o diagnóstico do Código de Falha 1962.
