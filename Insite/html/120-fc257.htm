Acionador da Válvula de Corte de Combustível do Motor - Sistema Mecânico NĂO Responde Corretamente ou Fora de Ajuste. A válvula de corte de combustível está engripada na posiçăo aberta ou com vazamento.

O motor funcionará despotenciado. O motor poderá continuar funcionando por um breve período quando a chave de igniçăo for desligada (OFF).
ISM11 CM876 SN - Válvula de Corte de Combustível do Motor

Descriçőes do Circuito

O módulo eletrônico de controle (ECM) fornece voltagem para a válvula de corte de combustível enquanto a chave de igniçăo está na posiçăo ON. Enquanto a chave de igniçăo está desligada (OFF), o ECM desliga a voltagem para a válvula de corte de combustível. A válvula de corte de combustível fecha e interrompe o fluxo de combustível para o sistema de combustível.

Localizaçăo do Componente

A válvula de corte de combustível está localizada próximo da bomba de combustível.

Condiçőes para Execuçăo do Diagnóstico

Este diagnóstico é executado assim que a chave de igniçăo é colocada na posiçăo OFF.

Condiçőes para Configurar os Códigos de Falha

A rotaçăo do motor é maior que um valor calibrável depois que a chave de igniçăo é desligada.

Açăo tomada quando o código de falha está ativo

  • O ECM acende a luz amarela de verificaçăo do motor (CHECK ENGINE) imediatamente quando o diagnóstico é executado e falha.

Condiçőes para Apagar o Código de Falha

O ECM apaga a luz âmbar de verificaçăo do motor (CHECK ENGINE) e o código de falha é imediatamente desativado quando o diagnóstico é executado com ęxito. A ferramenta eletrônica de serviço INSITE™ também pode ser usada para apagar o código de falha.

Dicas para a Oficina

Certifique-se de que a calibraçăo do ECM esteja correta. Verifique o histórico de revisőes de calibraçőes localizado no QuickServe™ Online para as correçőes aplicáveis ŕ calibraçăo armazenada no ECM. Se necessário, calibre o ECM.

Quando a chave de igniçăo é colocada na posiçăo OFF, o ECM desliga a voltagem fornecida para manter aberta a válvula de corte de combustível. O controle de combustível mantém a igniçăo dos injetores durante um curto período, ŕ medida que a rotaçăo do motor diminui. O ECM monitora a rotaçăo do motor utilizando o sensor de posiçăo do motor durante esse período. Se a rotaçăo do motor năo diminuir quando a chave de igniçăo for desligada (OFF), o ECM ativará este código de falha. Uma possível causa para este código de falha săo fios adicionais conectados na válvula de corte de combustível.

Consulte o diagnóstico do Código de Falha 257.
