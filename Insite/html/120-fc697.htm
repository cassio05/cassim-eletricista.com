Descries do Circuito

O mdulo eletrnico de controle (ECM) tem um sensor interno de temperatura que usado para medir a temperatura no interior do ECM.

Localizao do Componente

No se aplica

Condies para Execuo do Diagnstico

Este diagnstico executado continuamente quando a chave de ignio encontra-se na posio ON.

Condies para Configurar os Cdigos de Falha

O ECM detecta que o circuito do sensor da temperatura interna est em curto com uma fonte de voltagem alta.

Ao tomada quando o cdigo de falha est ativo

  • O ECM acende a luz mbar de verificao do motor (CHECK ENGINE) imediatamente quando o diagnstico executado e falha.

Condies para Apagar o Cdigo de Falha

  • Para validar o reparo, d partida no motor e deixe-o funcionar em marcha lenta durante 1 minuto.
  • O ECM apagar imediatamente a luz mbar de verificao do motor (CHECK ENGINE) depois que o diagnstico for executado com xito.
  • Este cdigo de falha tambm pode ser apagado com a ferramenta eletrnica de servio INSITE.

Dicas para a Oficina

Este sensor est localizado internamento no ECM. No possvel fazer o diagnstico deste cdigo de falha. Se este cdigo de falha for ativado, o ECM dever ser substitudo.
