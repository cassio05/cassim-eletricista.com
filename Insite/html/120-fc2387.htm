Circuito (Motor) do Acionador do Atuador do TGV - Sistema Mecânico Năo Responde Corretamente ou Fora de Ajuste. O controlador inteligente do TGV detectou limites de parada incorretos ou o TGV năo pode se mover para a posiçăo fechada.

O curso do TGV poderá ser limitado.
ISL8.9 CM2150 SN - Circuito do Acionador do Atuador do TGV

Descriçőes do Circuito

O turbocompressor de geometria variável (TGV) é ativado eletronicamente pelo atuador. O atuador do TGV é um dispositivo 'inteligente' que recebe informaçőes pelo datalink J1939 enviadas pelo módulo eletrônico de controle (ECM) do motor primário. O atuador do TGV realiza seus próprios diagnósticos e relata erros para o ECM do motor primário, através do datalink J1939.

Localizaçăo do Componente

O atuador do TGV está localizado no alojamento dos rolamentos do turbocompressor.

Condiçőes para Execuçăo do Diagnóstico

Este diagnóstico é executado continuamente quando a chave de igniçăo está na posiçăo ON ou quando o motor está funcionando.

Condiçőes para Configurar os Códigos de Falha

Este código de falha será ativado quando a posiçăo real do TGV năo for igual ŕ posiçăo comandada por mais de 10 segundos.

Açăo tomada quando o código de falha está ativo

  • O ECM acende a luz amarela de verificaçăo do motor (CHECK ENGINE) imediatamente quando o diagnóstico é executado e falha.
  • O atuador do TGV tentará continuamente a satisfazer ŕ posiçăo comandada mesmo quando a luz de verificaçăo do motor (CHECK ENGINE) estiver acesa.

Condiçőes para Apagar o Código de Falha

O ECM apagará imediatamente a luz de verificaçăo do motor (CHECK ENGINE) depois que o diagnóstico for executado com ęxito quando a chave de igniçăo for ligada (ON).

Dicas para a Oficina

Grandes vazamentos de pressăo de admissăo também podem ativar este código de falha.

O sistema de pós-tratamento deve ser inspecionado depois do reparo apropriado descrito neste diagrama de diagnóstico de código de falha, se aplicável. É possível que tenham ocorrido danos progressivos no sistema de pós-tratamento.

Consulte o diagnóstico do Código de Falha 2387.
