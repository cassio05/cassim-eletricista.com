Atuador do Solenóide do Injetor do Cilindro No. 4 - Fora de Calibraçăo. Os valores dos códigos de barras dos injetores estăo incorretos.

Nenhum quanto ao desempenho.
ISF2.8 CM2220 E/ISF2.8 CM2220 AN - Circuito do Atuador do Solenóide do Injetor do Cilindro No. 4

Descriçőes do Circuito

As válvulas-solenóide dos injetores săo atuadas pelo ECM para controlar a mediçăo do débito do combustível e a sincronizaçăo. Cada solenóide de injetor é conectado ao ECM através de um fio de alimentaçăo e um fio de retorno. Um pulso elétrico é enviado ao injetor pelo ECM através do fio de alimentaçăo e retorna ao ECM pelo fio de retorno depois de atuar o solenóide. As válvulas-solenóide elétricas săo normalmente abertas e săo fechadas somente por um pulso elétrico enviado pelo ECM durante a injeçăo e mediçăo de débito de combustível.

Localizaçăo do Componente

Os injetores passam pela parte superior da tampa dos balanceiros. Cada injetor tem um conector elétrico externo que é conectado diretamente no chicote do motor. O chicote do motor liga os injetores ao ECM.

Condiçőes para Execuçăo do Diagnóstico

O diagnóstico é executado continuamente enquanto o motor funciona.

Condiçőes para Configurar os Códigos de Falha

O ECM detecta que o valor do código de barras do injetor para o cilindro 4 năo foi inserido corretamente no ECM.

Açăo tomada quando o código de falha está ativo

O ECM acende a luz amarela de verificaçăo do motor (CHECK ENGINE) imediatamente quando o diagnóstico é executado e falha.

Condiçőes para Apagar o Código de Falha

O código de falha é apagado imediatamente depois que os valores de código de barras dos injetores săo inseridos com a ferramenta INSITE™.

Dę partida no motor e deixe-o funcionar em marcha lenta durante 1 minuto.

  • O ECM apaga a luz âmbar de verificaçăo do motor (CHECK ENGINE) e o código de falha é imediatamente desativado quando o diagnóstico é executado com ęxito.
  • Este código de falha também pode ser apagado com a ferramenta eletrônica de serviço INSITE™.

Dicas para a Oficina

Cada injetor de combustível tem um código de barras específico. Os dados do código de barras contęm informaçőes específicas sobre o injetor. Essas informaçőes de código de barras precisam ser inseridas no ECM usando-se a ferramenta eletrônica de serviço INSITE™. Se os valores de código de barras dos injetores estiverem incorretos, os valores corretos deverăo ser inseridos no ECM. O sintoma do motor para valores incorretos de códigos de barras pode incluir funcionamento irregular em marcha lenta. Para obter os valores de código de barras instalados na fábrica, acesse o QuickServe™ Online, informe o número de série correto do motor (ESN) e clique em Serviço e depois em Plaqueta de Dados do Motor. Os valores corretos dos códigos de barras dos injetores para esse ESN podem ser encontrados na seçăo de dados de identificaçăo do motor. Se o motor teve um reparo anterior envolvendo a troca dos injetores, os valores dos códigos de barras poderăo ser lidos diretamente na parte superior de cada injetor sem a necessidade de remover outros componentes. Use a ferramenta eletrônica de serviço INSITE™ para inserir os valores dos códigos de barras no ECM para desativar este código de falha.
