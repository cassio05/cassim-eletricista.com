Descriçőes do Circuito

O sensor de nível do tanque de fluido de escape de diesel é usado para monitorar o nível de fluido de escape de diesel no interior do tanque.

Localizaçăo do Componente

O sensor de nível de fluido de escape de diesel está localizado no tanque de fluido de escape de diesel. A localizaçăo do tanque e do sensor do nível da soluçăo depende do OEM.

Condiçőes para Execuçăo do Diagnóstico

Este diagnóstico é executado continuamente quando a chave de igniçăo encontra-se na posiçăo ON.

Condiçőes para Configurar os Códigos de Falha

Foi detectado nível baixo de fluido de escape de diesel no tanque de fluido do sistema de pós-tratamento.

Açăo tomada quando o código de falha está ativo

  • O ECM acende imediatamente a luz âmbar de verificaçăo do motor (CHECK ENGINE) e/ou a luz indicadora de falha (MIL) quando o diagnóstico é executado e falha.

Condiçőes para Apagar o Código de Falha

  • Para validar o reparo, dę partida no motor e deixe-o funcionar em marcha lenta durante 1 minuto.
  • O ECM apagará imediatamente a luz âmbar de verificaçăo do motor (CHECK ENGINE) depois que o diagnóstico for executado com ęxito.
  • O ECM apagará a luz indicadora de falha (MIL) depois de 3 ciclos consecutivos de igniçăo que o diagnóstico for executado com ęxito.
  • O código de falha e a luz MIL também podem ser apagados com a ferramenta eletrônica de serviço INSITE™.

Dicas para a Oficina

O Código de Falha 1671 é acionado quando o sensor de nível do fluido de escape de diesel detectar que o nível de fluido no tanque está baixo. Adicione fluido de escape de diesel no tanque. Este código de falha é somente uma advertęncia de nível baixo da soluçăo no tanque de fluido de escape de diesel.

Consulte o diagnóstico do Código de Falha 1671.
