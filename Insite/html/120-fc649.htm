Descriçőes do Circuito

O módulo eletrônico de controle (ECM) informa o motorista sobre um evento de manutençăo utilizando os limites do intervalo de troca de óleo lubrificante do recurso Monitor de Manutençăo.

Localizaçăo do Componente

Para obter a localizaçăo específica do componente, consulte o manual de serviços apropriado. Consulte o manual de diagnóstico de falhas e reparos do OEM para a localizaçăo específica do pórtico de enchimento de óleo.

Condiçőes para Execuçăo do Diagnóstico

Este diagnóstico é executado continuamente quando a chave de igniçăo encontra-se na posiçăo ON.

Condiçőes para Configurar os Códigos de Falha

Com base no ajuste programável no recurso Monitor de Manutençăo, o ECM detecta o intervalo de troca de óleo do motor e do filtro de óleo.

Açăo tomada quando o código de falha está ativo

  • O ECM acende imediatamente a luz âmbar de verificaçăo do motor (CHECK ENGINE) quando a condiçăo de manutençăo é excedida.

Condiçőes para Apagar o Código de Falha

O ECM apaga a luz âmbar de verificaçăo do motor (CHECK ENGINE) e o código de falha é desativado imediatamente após o recurso Monitor de Manutençăo ser reinicializado na ferramenta eletrônica de serviço INSITE™.

Dicas para a Oficina

Este código de falha somente se tornará ativo se o Monitor de Manutençăo estiver habilitado na ferramenta eletrônica de serviço INSITE™. Certifique-se de que o óleo e o filtro de óleo sejam trocados antes de apagar o código de falha.
