Circuito do Sensor da Temperatura Interna do ECM - Voltagem Abaixo da Normal ou com Voltagem Baixa. Detectada voltagem baixa de sinal ou um circuito aberto no sensor de temperatura interna do ECM.

Nenhum quanto ao desempenho.

Descriçőes do Circuito

O módulo eletrônico de controle (ECM) tem um sensor interno de temperatura que é usado para medir a temperatura no interior do ECM.

Localizaçăo do Componente

Condiçőes para Execuçăo do Diagnóstico

Este diagnóstico é executado continuamente quando a chave de igniçăo encontra-se na posiçăo ON.

Condiçőes para Configurar os Códigos de Falha

O ECM detecta que o circuito do sensor da temperatura interna está em curto com o massa.

Açăo tomada quando o código de falha está ativo

  • O ECM acende a luz âmbar de verificaçăo do motor (CHECK ENGINE) imediatamente quando o diagnóstico é executado e falha.

Condiçőes para Apagar o Código de Falha

  • Para validar o reparo, dę partida no motor e deixe-o funcionar em marcha lenta durante 1 minuto.
  • O ECM apagará imediatamente a luz âmbar de verificaçăo do motor (CHECK ENGINE) depois que o diagnóstico for executado com ęxito.
  • Este código de falha também pode ser apagado com a ferramenta eletrônica de serviço INSITE™.

Dicas para a Oficina

Este sensor está localizado internamento no ECM. Năo é possível fazer o diagnóstico deste código de falha. Se este código de falha for ativado, o ECM deverá ser substituído.

Consulte o diagnóstico do Código de Falha 698.
