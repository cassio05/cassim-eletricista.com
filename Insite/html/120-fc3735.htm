Interruptor do Governador Auxiliar do Motor - Taxa Anormal de Atualizaçăo. Perdido o sinal de comunicaçăo entre o módulo eletrônico de controle (ECM) e um outro dispositivo no datalink SAE J1939.

A rotaçăo do motor irá diminuir e permanecerá em marcha lenta.
ISB4.5, ISB6.7, ISD4.5 e ISD6.7 CM2150 SN/ISL8.9 CM2150 SN - Circuito de Multiplexaçăo

Descriçőes do Circuito

Dispositivos como controladores ABS, transmissőes automáticas, sistemas ASR, mostradores eletrônicos, sistemas eletrônicos de informaçőes, ferramentas eletrônicas de serviço e unidades eletrônicas de controle do veículo (VECUs) podem comunicar-se com o ECM via datalink SAE J1939. As mensagens enviadas por tais dispositivos săo recebidas pelo ECM e usadas para controlar o motor. O ECM também transmite as informaçőes para esses dispositivos pelo datalink SAE J1939.

Localizaçăo do Componente

A fiaçăo do datalink SAE J1939 e os dispositivos do SAE J1939 variam de acordo com as opçőes do OEM. Consulte o manual de serviços do OEM.

Condiçőes para Execuçăo do Diagnóstico

Este diagnóstico é executado continuamente quando a chave de igniçăo encontra-se na posiçăo ON.

Condiçőes para Configurar os Códigos de Falha

O ECM năo recebe uma mensagem válida via J1939 de um dispositivo multiplexado por mais de 1 segundo. O ECM verifica detecta uma condiçăo de circuito aberto ou de curto-circuito no chicote do datalink J1939.

Açăo tomada quando o código de falha está ativo

  • O ECM registra o código de falha imediatamente quando o diagnóstico é executado e falha.
  • O ECM usará um estado padrăo como entrada para o dispositivo multiplexado.

Condiçőes para Apagar o Código de Falha

  • Para validar o reparo, dę partida no motor e deixe-o funcionar em marcha lenta durante 1 minuto.
  • O status do código de falha mostrado na ferramenta eletrônica de serviço INSITE mudará para INATIVO imediatamente depois de o diagnóstico executar com ęxito.
  • O código de falha também pode ser apagado com a ferramenta eletrônica de serviço INSITE™.

Dicas para a Oficina

Possíveis causas deste código de falha:

  • Problemas elétricos com a fiaçăo do datalink SAE J1939 como circuitos aterrados, em curto-circuito ou abertos
  • O ECM (ou outro dispositivo no SAE J1939) interrompe as comunicaçőes enviando excesso de mensagens consecutivas
  • O dispositivo externo de controle do OEM falhou ou năo pode mais transmitir pelo datalink.
Consulte o diagnóstico do Código de Falha 3735.
