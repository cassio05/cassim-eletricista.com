Passos de Diagnóstico de Falha

PASSOS ESPECIFICAÇÕES
PASSO 1. Verifique se há códigos de falha do circuito do sensor.  
  PASSO 1A. Verifique se há códigos de falha ativos. Código de Falha 3142 ou 3143 ativos?
PASSO 2. Verifique se a operação do sistema de componentes a ar está correta.  
  PASSO 2A. Verifique o arrefecedor ar-ar, as braçadeiras, as mangueiras e a tubulação. Há vazamentos de ar ou braçadeiras soltas?
  PASSO 2B. Inspecione o compressor e as lâminas da turbina do turbocompressor. Algum dano encontrado nas lâminas do turbocompressor?
PASSO 3. Inspecione o sensor de temperatura na entrada do sistema de SCR.  
  PASSO 3A. Inspecione o sensor de temperatura na entrada do sistema de SCR. O sensor de temperatura na entrada de SCR está danificado?
PASSO 4. Verifique a calibração do ECM e apague os códigos de falha.  
  PASSO 4A. Verifique se existe disponível uma atualização de calibração para o ECM. Se houver uma atualização de calibração disponível para este código de falha, o ECM contém essa revisão ou uma posterior?
  PASSO 4B. Desative o código de falha. Código de falha inativo?


Passo Assistido 1 - Verifique se há códigos de falha do circuito do sensor.




Passo Assistido 1A - Verifique se há códigos de falha ativos.

Condições

  • Ligue a chave de ignição (ON).
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Ação

Verifique se há códigos de falha ativos.

  • Utilize a ferramenta eletrônica de serviço INSITE™ para ler os códigos de falha.
 

Código de Falha 3142 ou 3143 ativos?

Código de Falha 3142 ou 3143 ativos?
SIM NÃO
Sem Reparo Sem Reparo

Consulte o diagrama de falha do Código de Falha 3142 ou 3143.

Passo Assistido 2 - Verifique se a operação do sistema de componentes a ar está correta.




Passo Assistido 2A - Verifique o arrefecedor ar-ar, as braçadeiras, as mangueiras e a tubulação.

Condições

  • Desligue a chave de ignição (OFF).

Ação

Verifique se há vazamentos no arrefecedor ar-ar, nas mangueiras e nas tubulações.

  • Verifique se o torque das braçadeiras está correto.
  • Consulte o Procedimento 010-024 na Tabela de Procedimentos Associados.
 

Há vazamentos de ar ou braçadeiras soltas?

Há vazamentos de ar ou braçadeiras soltas?
SIM NÃO
Sem Reparo Sem Reparo

Ir para 2B




Passo Assistido 2B - Inspecione o compressor e as lâminas da turbina do turbocompressor.

Condições

  • Desligue a chave de ignição (OFF).
  • Remova as conexões de admissão e de escape do turbocompressor.

Ação

Inspecione o compressor do turbocompressor e as lâminas da turbina quanto a danos.

  • Consulte o Procedimento 010-033 na Tabela de Procedimentos Associados.
 

Algum dano encontrado nas lâminas do turbocompressor?

Algum dano encontrado nas lâminas do turbocompressor?
SIM NÃO

Substitua o turbocompressor.

Consulte o Procedimento 010-033 na Tabela de Procedimentos Associados.

Ir para 3A



Passo Assistido 3 - Inspecione o sensor de temperatura na entrada do sistema de SCR.




Passo Assistido 3A - Inspecione o sensor de temperatura na entrada do sistema de SCR.

Condições

  • Desligue a chave de ignição (OFF).
  • Desconecte do sistema de escape o sensor de temperatura na entrada de SCR.

Ação

Inspecione o sensor de temperatura na entrada do sistema de SCR.

  • Inspecione o sensor de temperatura na entrada de SCR quanto a danos ou sinais de temperatura excessivamente alta.
 

O sensor de temperatura na entrada de SCR está danificado?

O sensor de temperatura na entrada de SCR está danificado?
SIM NÃO

Substitua o sensor de temperatura na entrada do sistema de SCR.

Consulte o Procedimento 019-449 na Tabela de Procedimentos Associados.

Sem Reparo

Ir para 4A



Passo Assistido 4 - Verifique a calibração do ECM e apague os códigos de falha.




Passo Assistido 4A - Verifique se existe disponível uma atualização de calibração para o ECM.

Condições

  • Conecte todos os componentes.
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Ação

Compare o código do ECM e o número de revisão no ECM com as revisões de calibração relacionadas no Histórico de Revisões de Calibração do ECM para verificar se houve mudanças associadas a este código de falha.

  • Use a ferramenta eletrônica de serviço INSITE™ para localizar o código do ECM e o número de revisão existente no ECM. O código do ECM e o número de revisão encontram-se na seção Informações de Calibração, sob ID do Sistema e Plaqueta de Dados, na tela Recursos e Parâmetros.
 

Se houver uma atualização de calibração disponível para este código de falha, o ECM contém essa revisão ou uma posterior?

Se houver uma atualização de calibração disponível para este código de falha, o ECM contém essa revisão ou uma posterior?
SIM NÃO
Sem Reparo

Se necessário, calibre o ECM. Consulte o Procedimento 019-032 na Tabela de Procedimentos Associados.

Ir para 4B




Passo Assistido 4B - Desative o código de falha.

Condições

  • Conecte todos os componentes.
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Ação

Desabilite e apague o código de falha.

  • Funcione o motor de acordo com as "Condições para Apagar o Código de Falha" na seção de resumo deste procedimento de diagnóstico de falha.
 

Código de falha inativo?

Sem Reparo
