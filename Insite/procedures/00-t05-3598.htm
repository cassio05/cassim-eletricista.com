Passo Assistido 3 - Verifique a calibraçăo do ECM.




Passo Assistido 3A - Calibre o ECM.

Condiçőes

  • Conecte todos os componentes.
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Açăo

Compare o código do ECM e o número de revisăo no ECM em relaçăo ŕs revisőes de calibraçăo relacionadas no Histórico de Revisőes de Calibraçăo do ECM para verificar se houve mudanças associadas a este código de falha.

Use a ferramenta eletrônica de serviço INSITE™ para localizar o código do ECM e o número de revisăo existente no ECM. O código do ECM e o número de revisăo encontram-se na seçăo Informaçőes de Calibraçăo, sob ID do Sistema e Plaqueta de Dados, na tela Recursos e Parâmetros.

Calibre o ECM usando o maior número de revisăo disponível para a calibraçăo apropriada. Certifique-se de selecionar a calibraçăo correta para a aplicaçăo do motor. Consulte o Procedimento 019-032 na Tabela de Procedimentos Associados.

Depois de calibrar o ECM, funcione o motor de acordo com as "Condiçőes para Apagar o Código de Falha" na seçăo de resumo deste procedimento de diagnóstico.

Utilize a ferramenta eletrônica de serviço INSITE™ para verificar os códigos de falha.

O Código de Falha 3598 foi ativado novamente?

Substitua o controlador de fluido de escape de diesel do sistema de pós-tratamento. Consulte o Procedimento 011-057 na Tabela de Procedimentos Associados.

A calibraçăo do ECM corrigiu o problema.
