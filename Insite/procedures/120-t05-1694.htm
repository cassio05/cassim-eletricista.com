Passo Assistido 1 - Verifique os códigos de falha.




Passo Assistido 1A - Verifique se há códigos de falha ativos.

Condiçőes

  • Ligue a chave de igniçăo (ON).
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Açăo

Verifique se há códigos de falha ativos.

  • Utilize a ferramenta eletrônica de serviço INSITE™ para ler os códigos de falha.
  • Faça primeiro o diagnóstico de outros códigos de falha ativos.
  • Todos os outros códigos ativos devem ser resolvidos antes de ser iniciado este diagnóstico de falha.

Passo Assistido 1B - Verifique o sistema de escape.

Condiçőes

  • Remova o sensor de NOx na saída do sistema de pós-tratamento. Consulte o Procedimento 019-451 na Tabela de Procedimentos Associados.

Açăo

Verifique o sistema de escape quanto a:

  • Vazamentos no sistema de escape entre o turbocompressor e o sistema do catalisador de SCR.
  • Verifique o sensor de NOx na saída do sistema de pós-tratamento quanto ŕ instalaçăo correta. Consulte o Procedimento 019-451 na Tabela de Procedimentos Associados.
  • Verifique se há depósitos de carbono na ponta do sensor de NOx na saída do sistema de pós-tratamento. Consulte o Procedimento 019-451 na Tabela de Procedimentos Associados.
  • Certifique-se de que o número de peça do sensor de NOx na saída do sistema de pós-tratamento esteja correto para o modelo do motor.
 

O sensor de NOx na saída do sistema de pós-tratamento está instalado corretamente e o sistema de escape está livre de vazamentos?

O sensor de NOx na saída do sistema de pós-tratamento está instalado corretamente e o sistema de escape está livre de vazamentos?
SIM NĂO
Sem Reparo

Repare o sistema de escape ou verifique a instalaçăo do sensor de NOx.

Consulte o Procedimento 019-451 na Tabela de Procedimentos Associados.

Consulte o manual de serviços do OEM para a instalaçăo correta do sistema de escape.

Passo Assistido 1C - Verifique se a unidade de controle de dosagem do catalisador de SCR funciona corretamente.

Condiçőes

  • Ligue a chave de igniçăo (ON).
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Açăo

Verifique a unidade de controle de dosagem do catalisador de SCR.

  • Use a ferramenta eletrônica de serviço INSITE™ para executar o Teste de Anulaçăo da Bomba Dosadora de Uréia, no menu Testes de Diagnóstico do ECM.

Consulte o procedimento abaixo para obter instruçőes e os critérios de aprovaçăo/reprovaçăo de testes para sistemas de dosagem de SCR assistidos a ar. Consulte o Procedimento 019-440 na Tabela de Procedimentos Associados.

Consulte o procedimento abaixo para obter instruçőes e os critérios de aprovaçăo/reprovaçăo de testes para sistemas de dosagem de SCR sem ar. Consulte o Procedimento 011-058 na Tabela de Procedimentos Associados.

Nota: O injetor de soluçăo do catalisador deve ser removido e colocado em um recipiente de mediçăo para a execuçăo deste teste. Certifique-se de que o injetor do sistema de pós-tratamento năo esteja obstruído e que o fluido de escape de diesel esteja fluindo de todos os orifícios.
 

A unidade de controle de dosagem do catalisador de SCR funciona corretamente durante os testes de anulaçăo do INSITE™?

A unidade de controle de dosagem do catalisador de SCR funciona corretamente durante os testes de anulaçăo do INSITE™?
SIM NĂO
Sem Reparo

Substitua a unidade de controle de dosagem do catalisador de SCR.

Consulte o procedimento abaixo para sistemas dosadores de SCR assistidos a ar. Consulte o Procedimento 019-440 na Tabela de Procedimentos Associados.

Consulte o procedimento abaixo para sistemas dosadores de SCR sem ar. Consulte o Procedimento 011-058 na Tabela de Procedimentos Associados.

Passo Assistido 2A - Verifique a calibraçăo do ECM e apague os códigos de falha.




Passo Assistido 2A - Verifique se existe disponível uma atualizaçăo de calibraçăo para o ECM.

Condiçőes

  • Conecte todos os componentes.
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Açăo

Compare o código do ECM e o número de revisăo no ECM em relaçăo ŕs revisőes de calibraçăo relacionadas no Histórico de Revisőes de Calibraçăo do ECM para verificar se houve mudanças associadas a este código de falha.

  • Use a ferramenta eletrônica de serviço INSITE™ para localizar o código do ECM e o número de revisăo existente no ECM. O código do ECM e o número de revisăo encontram-se na seçăo Informaçőes de Calibraçăo, sob ID do Sistema e Plaqueta de Dados, na tela Recursos e Parâmetros.
 

Se houver uma atualizaçăo de calibraçăo disponível para este código de falha, o ECM contém essa revisăo ou uma posterior?

Se houver uma atualizaçăo de calibraçăo disponível para este código de falha, o ECM contém essa revisăo ou uma posterior?
