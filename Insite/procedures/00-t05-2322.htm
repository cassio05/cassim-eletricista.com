Passos de Diagnóstico de Falha

PASSOS ESPECIFICAÇŐES
PASSO 1. Verifique os códigos de falha.  
  PASSO 1A. Verifique se há um código de falha ativo. O Código de Falha 778 está ativo ou existem altas contagens de códigos de falha inativos?
PASSO 2. Verifique o circuito do sensor secundário de rotaçăo (sensor da posiçăo do eixo comando de válvulas).  
  PASSO 2A. Inspecione o sensor secundário de rotaçăo do motor (sensor da posiçăo do eixo comando de válvulas) e os pinos do conector. Pinos sujos ou danificados?
  PASSO 2B. Inspecione o chicote do motor. Isolamento da fiaçăo danificado?
  PASSO 2C. Inspecione o circuito do sensor secundário de rotaçăo (sensor do eixo comando de válvulas) no conector do chicote do motor. Pinos sujos ou danificados?
  PASSO 2D. Inspecione o chicote do motor. Isolamento da fiaçăo danificado?
PASSO 3. Teste o veículo.  
  PASSO 3A. Faça o teste de rodagem para determinar se existe condiçăo de falha. O Código de Falha 2322 tornou-se ativo durante o funcionamento do motor?
  PASSO 3B. Inspecione a folga de ar e a roda de referęncia. A folga de ar está dentro das especificaçőes?
  PASSO 3C. Manipule o chicote do motor para testar a existęncia de uma condiçăo de falha intermitente no chicote. O status do Código de Falha 2322 muda de inativo para ativo durante a flexăo do chicote do motor?
PASSO 4. Apague os códigos de falha.  
  PASSO 4A. Verifique se existe disponível uma atualizaçăo de calibraçăo para o ECM. Se houver uma atualizaçăo de calibraçăo disponível para este código de falha, o ECM contém essa revisăo ou uma posterior?
  PASSO 4B. Desative o código de falha. Código de falha inativo?


Passo Assistido 1 - Verifique os códigos de falha.




Passo Assistido 1A - Verifique se há um código de falha ativo.

Condiçőes

  • Ligue a chave de igniçăo (ON).
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Açăo

Verifique se há um código de falha ativo.

  • Utilize a ferramenta eletrônica de serviço INSITE™ para ler os códigos de falha.
 

O Código de Falha 778 está ativo ou existem altas contagens de códigos de falha inativos?

Nota: Se o Código de Falha 778 tiver um alto número de contagens inativas, provavelmente existe um problema de circuito intermitente (curto ou aberto) no circuito do sensor primário de rotaçăo.

O Código de Falha 778 está ativo ou existem altas contagens de códigos de falha inativos?

Sem Reparo Sem Reparo

Consulte o diagrama de diagnóstico de falha do Código de Falha 778

Passo Assistido 2 - Verifique o circuito do sensor secundário de rotaçăo (sensor da posiçăo do eixo comando de válvulas).




Passo Assistido 2A - Inspecione o sensor secundário de rotaçăo do motor (sensor da posiçăo do eixo comando de válvulas) e os pinos do conector.

Condiçőes

  • Desligue a chave de igniçăo (OFF).
  • Desconecte do chicote do motor o sensor secundário de rotaçăo.

Açăo

Inspecione os pinos dos conectores do chicote do motor e do atuador quanto a:

  • Conector solto
  • Pinos corroídos
  • Pinos tortos ou quebrados
  • Pinos torcidos para trás ou expandidos
  • Umidade dentro ou sobre o conector
  • Vedaçőes do conector danificadas ou faltando
  • Sujeira ou resíduos nos pinos do conector
  • Capa do conector quebrada
  • Dano na blindagem do fio
  • Dano na guia de trava do conector.

Use o seguinte procedimento de técnicas gerais de inspeçăo. Consulte o Procedimento 019-361 na Seçăo 19.

 

Pinos sujos ou danificados?

Pinos sujos ou danificados?
SIM NĂO

Foi detectada uma conexăo defeituosa no sensor ou no conector do chicote.

Limpe o conector e os pinos.

Se danificados, repare o chicote, o conector ou os pinos, se possível.

Consulte o Procedimento 019-043 na Tabela de Procedimentos Associados.

Sem Reparo

Passo Assistido 3A - Faça o teste de rodagem para determinar se existe condiçăo de falha.

Condiçőes

  • Conecte o sensor secundário de rotaçăo (sensor da posiçăo do eixo comando de válvulas) no chicote do motor.

Com o motor funcionando, acelere o motor várias vezes.

  • Utilize a ferramenta eletrônica de serviço INSITE™ para observar os códigos de falha.

Passo Assistido 3B - Inspecione a folga de ar e a roda de referęncia.

Condiçőes

  • Desligue a chave de igniçăo (OFF).

Açăo

Inspecione a folga de ar.

  • Remova sensor secundário de rotaçăo do motor (sensor da posiçăo do eixo comando de válvulas) da carcaça das engrenagens e meça a folga de ar em vários pontos na rotaçăo da árvore de manivelas.

Consulte o Procedimento 019-363 na Tabela de Procedimentos Associados.

 

A folga de ar está dentro das especificaçőes?

  • Folga de Ar Mínima: 0.96 mm [0.038 in]
  • Folga de Ar Máxima: 2.25 mm [0.089 in]
  • Excentricidade Máxima da Roda de Referęncia: 0.5 mm [0.02 in]

A folga de ar está dentro das especificaçőes?

  • Folga de Ar Mínima: 0.96 mm [0.038 in]
  • Folga de Ar Máxima: 2.25 mm [0.089 in]
  • Excentricidade Máxima da Roda de Referęncia: 0.5 mm [0.02 in]
SIM NĂO
Sem Reparo

Substitua ou calce o sensor se a folga de ar estiver fora de especificaçăo. Substitua a roda de referęncia se sua excentricidade for excessiva.

Passo Assistido 3C - Manipule o chicote do motor para testar a existęncia de uma condiçăo de falha intermitente no chicote.

Condiçőes

  • Ligue a chave de igniçăo (ON).
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Açăo

Verifique se há uma condiçăo de falha intermitente.

Utilize a ferramenta eletrônica de serviço INSITE™ para monitorar o status do Código de Falha 2322.

  • Dę partida no motor e deixe-o funcionar em marcha lenta.
  • Começando pelo conector do ECM, flexione manualmente o chicote do motor. Continue a flexionar a partir do conector do ECM em direçăo ao conector do sensor da rotaçăo do motor.
 

O status do Código de Falha 2322 muda de inativo para ativo durante a flexăo do chicote do motor?

O status do Código de Falha 2322 muda de inativo para ativo durante a flexăo do chicote do motor?

Consulte o Procedimento 019-043 na Tabela de Procedimentos Associados.

Passo Assistido 4 - Verifique a calibraçăo do ECM e apague os códigos de falha.




Passo Assistido 4A - Verifique se existe disponível uma atualizaçăo de calibraçăo para o ECM.

Condiçőes

  • Conecte todos os componentes.
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Açăo

Compare o código do ECM e o número de revisăo no ECM com as revisőes de calibraçăo relacionadas no Histórico de Revisőes de Calibraçăo do ECM para verificar se houve mudanças associadas a este código de falha.

  • Use a ferramenta eletrônica de serviço INSITE™ para localizar o código do ECM e o número de revisăo existente no ECM. O código do ECM e o número de revisăo encontram-se na seçăo Informaçőes de Calibraçăo, sob ID do Sistema e Plaqueta de Dados, na tela Recursos e Parâmetros.
 

Se houver uma atualizaçăo de calibraçăo disponível para este código de falha, o ECM contém essa revisăo ou uma posterior?

Se houver uma atualizaçăo de calibraçăo disponível para este código de falha, o ECM contém essa revisăo ou uma posterior?
SIM NĂO
Sem Reparo
