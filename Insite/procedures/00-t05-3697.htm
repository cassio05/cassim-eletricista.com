Passo Assistido 2 - Recalibre o ECM.




Passo Assistido 2A - Recalibre o ECM.

Condiçőes

  • Conecte todos os componentes.
  • Ligue a chave de igniçăo (ON).
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Açăo

Use a ferramenta eletrônica de serviço INSITE™ para recalibrar o ECM com a calibraçăo mais recente. Consulte as revisőes de calibraçăo relacionadas no Histórico de Calibraçőes do ECM para localizar a última calibraçăo disponível para o motor em questăo.

  • Use a ferramenta eletrônica de serviço INSITE™ para localizar o código do ECM e o número de revisăo no ECM. O código e o número de revisăo do ECM encontram-se na seçăo Informaçőes de Calibraçăo, sob ID do Sistema e Plaqueta de Dados, na tela Recursos e Parâmetros.
 

O Código de Falha 3697 permanece ativo depois da calibraçăo do ECM?

O Código de Falha 3697 permanece ativo depois da calibraçăo do ECM?
SIM NĂO

Ligue para obter autorizaçăo para substituir o ECM. Depois de receber a autorizaçăo, substitua o ECM. Consulte o Procedimento 007-031 na Seçăo 19.

Passo Assistido 3 - Verifique a calibraçăo do ECM e apague os códigos de falha.




Passo Assistido 3A - Verifique a calibraçăo do ECM e apague os códigos de falha.

Condiçőes

  • Conecte todos os componentes
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Açăo

Compare o código do ECM e o número de revisăo no ECM com as revisőes de calibraçăo relacionadas no Histórico de Revisőes de Calibraçăo do ECM para verificar se houve mudanças associadas a este código de falha.

  • Use a ferramenta eletrônica de serviço INSITE™ para localizar o código do ECM e o número de revisăo existente no ECM. O código do ECM e o número de revisăo encontram-se na seçăo Informaçőes de Calibraçăo, sob ID do Sistema e Plaqueta de Dados, na tela Recursos e Parâmetros.
 

Se houver uma atualizaçăo de calibraçăo disponível para este código de falha, o ECM contém essa revisăo ou uma posterior?

Se houver uma atualizaçăo de calibraçăo disponível para este código de falha, o ECM contém essa revisăo ou uma posterior?
SIM NĂO
Sem Reparo

Se necessário, calibre o ECM. Consulte o Procedimento 019-032 na Tabela de Procedimentos Associados.
