Passo Assistido 1 - Verifique a calibraçăo do ECM.




Passo Assistido 1A - Verifique a calibraçăo do ECM.

Condiçőes

  • Desligue a chave de igniçăo (OFF).
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Açăo

Verifique a calibraçăo do ECM e certifique-se de que a mesma é a correta para a CPL do motor.

  • Certifique-se de que a voltagem de calibraçăo para o código do ECM é a correta para a aplicaçăo. Uma calibraçăo de 24 volts descarregada em um sistema de 12 volts pode causar este código de falha.
 

O código do ECM é o correto para a CPL do motor?

O código do ECM é o correto para a CPL do motor?
SIM NĂO

Foi detectado que o atuador do TGV é incorreto para a aplicaçăo da voltagem.

Substitua o atuador do TGV.

Consulte o Procedimento 010-134 na Tabela de Procedimentos Associados.

Descarregue o código do ECM correto para a CPL do motor.

Passo Assistido 2A - Verifique se existe disponível uma atualizaçăo de calibraçăo para o ECM.

Condiçőes

  • Conecte todos os componentes.
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Açăo

Compare o código do ECM e o número de revisăo no ECM em relaçăo ŕs revisőes de calibraçăo relacionadas no Histórico de Revisőes de Calibraçăo do ECM para verificar se houve mudanças associadas a este código de falha.

  • Use a ferramenta eletrônica de serviço INSITE™ para localizar o código do ECM e o número de revisăo existente no ECM. O código do ECM e o número de revisăo encontram-se na seçăo Informaçőes de Calibraçăo, sob ID do Sistema e Plaqueta de Dados, na tela Recursos e Parâmetros.
