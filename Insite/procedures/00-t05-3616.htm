Código de Falha 2387 ativo ou altas contagens de códigos inativos?

Código de Falha 2387 ativo ou altas contagens de códigos inativos?
SIM NĂO
Sem Reparo

O Código de Falha 3616 é um código informativo que indica que o pulverizador deslizante do TGV năo foi capaz de se mover para a posiçăo comandada quando a chave de igniçăo foi ligada (posiçăo ON). Năo é necessário fazer o diagnóstico deste código de falha. O código é apenas uma indicaçăo de que a potęncia do motor poderá ser limitada ou que a regeneraçăo do sistema de pós-tratamento pode ser ineficaz porque o pulverizador deslizante do TGV está engripado.

Passo Assistido 2 - Verifique a calibraçăo do ECM e apague os códigos de falha.




Passo Assistido 2A - Verifique se existe disponível uma atualizaçăo de calibraçăo para o ECM.

Condiçőes

  • Conecte todos os componentes
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Açăo

Compare o código do ECM e o número de revisăo no ECM com as revisőes de calibraçăo relacionadas no Histórico de Revisőes de Calibraçăo do ECM para verificar se houve mudanças associadas a este código de falha.

  • Use a ferramenta eletrônica de serviço INSITE™ para localizar o código do ECM e o número de revisăo existente no ECM. O código do ECM e o número de revisăo encontram-se na seçăo Informaçőes de Calibraçăo, sob ID do Sistema e Plaqueta de Dados, na tela Recursos e Parâmetros.
 

Se houver uma atualizaçăo de calibraçăo disponível para este código de falha, o ECM contém essa revisăo ou uma posterior?
