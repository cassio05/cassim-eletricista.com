Passos de Diagnóstico de Falha

PASSOS ESPECIFICAÇŐES
PASSO 1. Verifique os códigos de falha.  
  PASSO 1A. Verifique se há códigos de falha ativos associados ao turbocompressor. Códigos de Falha 2634, 2449 ou 2387 ativos?
  PASSO 1B. Verifique se há códigos de falha inativos. Código de Falha 295 ou 2973 ativos?
PASSO 2. Verifique as leituras do sensor.  
  PASSO 2A. Verifique a precisăo do sensor de pressăo do ar ambiente. A leitura da ferramenta eletrônica de serviço INSITE™ está dentro da faixa de 10 cm-Hg [4 pol-Hg] da pressăo barométrica local?
  PASSO 2B. Verifique a precisăo do sensor de pressăo no coletor de admissăo. A leitura da pressăo no coletor de admissăo é menor que 10 cm-Hg [4 pol-Hg]?
PASSO 3. Verifique a calibraçăo do ECM e apague os códigos de falha.  
  PASSO 3A. Verifique se existe disponível uma atualizaçăo de calibraçăo para o ECM. Se houver uma atualizaçăo de calibraçăo disponível para este código de falha, o ECM contém essa revisăo ou uma posterior?
  PASSO 3B. Desative o código de falha. Código de falha inativo?

Passo Assistido 3 - Verifique a calibraçăo do ECM e apague os códigos de falha.




Passo Assistido 3A - Verifique se existe disponível uma atualizaçăo de calibraçăo para o ECM.

Condiçőes

  • Conecte todos os componentes.
  • Conecte a ferramenta eletrônica de serviço INSITE™.

Açăo

Compare o código do ECM e o número de revisăo no ECM com as revisőes de calibraçăo relacionadas no Histórico de Revisőes de Calibraçăo do ECM para verificar se houve mudanças associadas a este código de falha.

  • Use a ferramenta eletrônica de serviço INSITE™ para localizar o código do ECM e o número de revisăo existente no ECM. O código do ECM e o número de revisăo encontram-se na seçăo Informaçőes de Calibraçăo, sob ID do Sistema e Plaqueta de Dados, na tela Recursos e Parâmetros.
 

Se houver uma atualizaçăo de calibraçăo disponível para este código de falha, o ECM contém essa revisăo ou uma posterior?

Se houver uma atualizaçăo de calibraçăo disponível para este código de falha, o ECM contém essa revisăo ou uma posterior?

Sem Reparo
